Car Rental System :  
1 - Program  
- CarRentalSystem.py est l'application  
- Tout les autres fichiers dans Programm doivent être dans le même dossier que CarRentalSystem.py
- L'application a un Login  
  - User : root
  - Password : root  
- Fonction de l'application :
  - Louer des voitures
  - Ajouter des clients et des véhicules
  - Permet à un client de commander un véhicule
  - Permet au personnel à voir les rapports sur les clients et les véhicules
